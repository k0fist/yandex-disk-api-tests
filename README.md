# Yandex Disk API Tests

Проект с автотестами для REST API Яндекс Диска.

В проекте реализованы:
- интеграционные тесты реального REST API Яндекс Диска;
- unit-тесты API-клиента с мокированием HTTP-запросов через `responses`;
- контрактная проверка JSON-ответов через `jsonschema`;
- позитивные и негативные сценарии;
- подготовка тестовых данных через `pytest fixtures`;
- очистка тестовых ресурсов после выполнения тестов.

## Что проверяется

### Информация о диске
- получение общей информации о диске пользователя;
- проверка структуры ответа через JSON Schema;
- проверка корректности значений `total_space` и `used_space`.

### Загрузка файлов
- получение ссылки для загрузки файла;
- загрузка временного файла на Яндекс Диск;
- проверка, что загруженный файл реально появился на диске;
- проверка ошибки при запросе upload-ссылки без обязательного параметра `path`;
- проверка сценария загрузки существующего файла без `overwrite`.

### Копирование файлов
- копирование файла на Яндекс Диске;
- проверка, что копия файла была создана;
- проверка ошибки при копировании несуществующего файла.

### Удаление ресурсов
- удаление загруженного файла;
- проверка удаления файла через запрос метаинформации;
- проверка ошибки при удалении несуществующего ресурса.

### Unit-тесты API-клиента
- проверка формирования URL;
- проверка установки авторизационных заголовков;
- проверка корректности запроса на получение upload-ссылки через mock.


## Стек

- python-dotenv

## Установка проекта, команды для git bash
1. Создания виртуального окружения - python -m venv venv
2. Активация виртуального окружения - source venv/Scripts/activate
3. Установка зависимостей - pip install -r requirements.txt
4. Создать файл с переменными окружений (.env) по шаблону файла .env.example и поместить его в корень проекта

## Запуск тестов
python -m pytest -v
